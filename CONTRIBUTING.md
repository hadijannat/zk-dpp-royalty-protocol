# Contributing

Thanks for your interest in contributing.

## Guidelines

- Open an issue for substantial changes before starting work.
- Keep changes focused and include tests when applicable.
- Follow existing code style and conventions.

## Development

```bash
pnpm install
pnpm rust:build
pnpm circuits:test
```

## Pull Requests

- Describe the problem and the solution clearly.
- Include screenshots or logs when relevant.
- Ensure CI passes before requesting review.
